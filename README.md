# Digital-Integrated-Technologies
Digital Integrated Technologies is a subject at the Technical University of Sofia, focused on the simulation of digital logic elements such as AND, NAND, OR, NOR,  XOR, XNOR gates using Microwind2.

---

##  Notes

* Implemented logic gates using NMOS and PMOS transistors
* Designed inverters as core circuit elements
* Used Metal1, Metal2, and polysilicon layers for layout design
* Added contacts between layers to ensure proper electrical connections
* Simulated and analyzed circuits using Microwind2
